## Usage

Use Divider to reinforce content hierarchy.

- Separate TabBar from FilterBar.

## Types

### Default

The default Divider is horizontal.

```tsx live
<Divider />
```

### Vertical

Vertical divider.

```tsx live
<div style={{ height: 256 }}>
  <Divider isVertical={true} />
</div>
```
